# Resource Insight Legacy Business Documentation

This folder documents the current business behavior embodied by `resource-insight`, which operates as SETA's internal review and performance-management system. The focus is cloning fidelity: what the system does for the business today, who uses it, what rules govern it, and what must not change if the behavior is reimplemented elsewhere.

## Documents

- `01-business-overview.md`: What the system is, what problem it solves, who uses it, where its boundaries are, and which conclusions are confirmed versus inferred.
- `02-capabilities-and-use-cases.md`: The major business capabilities and the user-facing use cases they support.
- `03-workflows-rules-and-domain.md`: The core operating mechanics: workflows, rules, permissions, entity meanings, and business state changes.
- `04-clone-critical-understanding.md`: The clone checklist. This is the most important document if a team must reproduce current business behavior faithfully.

## Recommended Reading Order

1. `01-business-overview.md`
2. `02-capabilities-and-use-cases.md`
3. `03-workflows-rules-and-domain.md`
4. `04-clone-critical-understanding.md`

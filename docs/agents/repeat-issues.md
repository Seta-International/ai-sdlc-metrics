# Agent Authoring — Repeat Issues

Destination for quarterly override audit findings. Each entry captures a cycle's findings and action items.

## How to Use

Run `bun run tools/lint/agent-authoring/audit-overrides.ts` quarterly.
Rules with override counts above the threshold (default: 3) surface here for owner review.
Outcomes: rule threshold change, rule description update, or rule removal.

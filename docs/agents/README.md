# Agent Memory Logs

This folder stores repo-specific agent lessons that should not live in `AGENTS.md`.

- `critical-decisions.md` — durable decisions that change future work.
- `repeat-issues.md` — recurring failures, confusion, or corrections.

Before non-trivial agent work, check both files.

Keep entries short and searchable:

- `Area:` where it applies
- `Context:` what triggered it
- `Decision:` or `Issue:` the takeaway
- `Action:` what changed

If an entry becomes wrong, update it explicitly.

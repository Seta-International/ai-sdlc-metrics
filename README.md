# Seta OS

> Agent-first runtime for the modern SaaS ERP.

Seta OS is Seta's open-source agent platform — a type-safe, framework-free agent kernel plus the primitives (auth, OAuth, observability, multi-tenant DB, channel adapters) for mounting agents into chat, ERP, and web surfaces.

**Status:** pre-1.0, active development. APIs may change without notice. Not yet published to npm.

## Stack

Node 24 · pnpm · Turborepo · TypeScript (strict) · Hono · Zod · Drizzle ORM · PostgreSQL + pgvector · OpenTelemetry · Vitest · Biome.
